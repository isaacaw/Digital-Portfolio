Veltvet Oasis

End-to-end UX journey from user research to interactive prototype.

  1. Home
  2. Projects
  3. UX Design

Project Objective

This UX case study focused on improving the user journey for a mobile-first service booking app. The main goal was to identify user pain points and redesign a simplified, user-centric experience for new and returning users.

Vision

To create a seamless and intuitive design backed by user research, focused on accessibility, responsiveness, and clarity in interactions. The solution aimed to reduce drop-offs and increase engagement through clean UX practices and a user-friendly interface.

Project Responsibilities

  • Owned the entire UX process from user research to high-fidelity prototyping.
  • Conducted user interviews, created personas, and mapped out the user journey.
  • Designed wireframes and interactive prototypes in Figma based on usability insights.
  • Performed usability testing and iterated based on real user feedback.
  • Ensured the final designs met accessibility, clarity, and responsiveness standards.

Challenges & Key Learnings

Designing for diverse user needs required a balance of functionality, simplicity, and aesthetics.

I learned how to translate abstract user feedback into actionable design decisions, how to prioritize features based on user needs, and how to create consistent design systems that adapt to different use cases.

Testing with real users also gave me valuable insights into iterative design thinking and rapid prototyping.

Through this project, I gained hands-on experience with:

  • User research and persona development
  • Wireframing and prototyping with Figma
  • Usability testing and iteration
  • Responsive and accessible UI design
