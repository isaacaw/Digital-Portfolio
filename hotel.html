Verde Vista Hotel

A seamless hotel website with calendar-based room booking and cart functionality.

  1. Home
  2. Projects
  3. Verde Vista Hotel

Project Objective

This hotel website project was designed to offer users a smooth and intuitive hotel booking experience. It allows users to view available room types, select their desired check-in and check-out dates using a calendar, and add rooms to their cart before checking out.

Vision

Our vision was to simulate a real-world e-booking system, incorporating both frontend design and backend functionality to support reservation workflows, date management, and pricing logic.

My Role in the Project

  • Designed the user interface for the facilities and room listing pages.
  • Implemented a calendar feature allowing users to select booking dates visually.
  • Created a cart function that enables users to add room bookings and review them before confirming.
  • Connected frontend pages to backend logic using Flask, enabling real-time room listing updates and dynamic interactions.
  • Developed admin-side functions to add, edit, and remove hotel rooms, with updates instantly reflected on the customer-facing pages.

Challenges & Key Learnings

Implementing the calendar-based booking feature required robust date validation and handling scenarios like overlapping or invalid date ranges.

I connected the frontend booking interface to Flask routes, storing and updating room data dynamically. I also managed user-specific booking sessions using Flask’s built-in session handling, ensuring a smooth cart experience.

The project reinforced best practices for routing, form handling (POST/GET), and linking frontend inputs with backend storage—all while keeping the interface user-friendly.

Through this project, I gained hands-on experience with:

  • Flask framework for routing and backend logic
  • Validating calendar-based date selections
  • Managing booking sessions using Flask
  • Integrating UI interactions with backend data updates
